# ML_Based_Buyer_Segmentation
# 🏠 Machine Learning Based Buyer Segmentation and Investment Profiling for Real Estate Market Intelligence

## 📌 Project Overview

This project focuses on identifying different categories of real estate buyers using Machine Learning techniques. By analyzing customer demographics, investment preferences, loan behavior, satisfaction scores, and property transaction data, the system groups buyers into meaningful segments.

The project also includes an interactive Streamlit dashboard that provides real-time insights into buyer behavior, geographic distribution, investment trends, and customer segmentation.

---

## 🎯 Objectives

* Analyze buyer demographics and property transaction data.
* Perform Exploratory Data Analysis (EDA).
* Preprocess and clean the dataset.
* Apply K-Means Clustering for buyer segmentation.
* Develop an interactive Streamlit dashboard.
* Generate business insights and recommendations for stakeholders.
